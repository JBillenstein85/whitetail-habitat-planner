
# Whitetail Habitat Planner

This repository contains structured data and tools for planning and optimizing whitetail deer habitats using a combination of AI insights, video-based expert knowledge, and geospatial imagery.

## Contents

- `data/whitetail_habitat_techniques_full.csv`: Structured CSV file containing habitat screening techniques extracted from expert videos.
- `database/whitetail_habitat.db`: SQLite database version of the dataset.
- `database/schema.sql`: SQL schema definition for creating and populating the `Techniques` table.
- `README.md`: Project overview and data description.

## Goals

- Automate whitetail habitat planning using AI and structured habitat techniques.
- Integrate video insights with spatial imagery (drone, 360Â°, satellite) to generate land development strategies.
- Enable landowners and hunters to improve habitat efficiency and effectiveness.
